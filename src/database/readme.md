# PackWise Database

Owner: Cristine

## Module
Packaging Recommendation & Cost Optimization

## Tables

### attachment_methods
Stores available packaging attachment methods.

### attachment_rules
Stores recommendation rules based on complexity score.

### recommendations
Stores generated packaging recommendations.

### cost_analysis
Stores packaging cost calculations and savings analysis.

## Database Flow

Product Analysis (Shanty)
↓
Recommendations (Cristine)
↓
Cost Analysis (Cristine)
↓
Risk Prediction (Nasywa)
